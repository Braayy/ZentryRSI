# ZentryRSI - Zentry Rest Interpreter

#### Maven install:

Download the source and execute `mvn install`
```xml
<dependencies>
  <dependency>
    <groupId>network.zentry</groupId>
    <artifactId>rsi</artifactId>
    <version>1.0</version>
    <scope>compile</scope>
  </dependency>
</dependencies>
```
